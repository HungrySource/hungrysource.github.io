You can edit, review and search your log entries in this screen.

  • Tap on the time button on the right hand to review entries in a day calendar format.
  • Tap an entry -> Statistics to see summary of that entry.
  • Tap an entry -> "Start this now" to start a past activity without typing.

"How to edit a log entry?"
Tap on an entry -> Rename.

"How to change the start time of a log entry?"
Tap on its time to change it.

"How to search?"
Scroll up, tap on search box and start typing.
